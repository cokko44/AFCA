# Contributing to ACFA

Thank you for your interest in ACFA.

Adaptive Cognitive Field Architecture (ACFA) is an open research specification for formal cognitive state transition systems.

## Principles

Contributions should preserve:

- Mathematical consistency
- Deterministic behavior where specified
- Benchmark reproducibility
- Documentation quality
- Test coverage

## Development Workflow

1. Fork the repository
2. Create a feature branch
3. Implement your changes
4. Add tests
5. Run all tests
6. Submit a Pull Request

## Coding Style

- Python 3.11+
- Type hints preferred
- Small modular functions
- Pure functions whenever possible

## Documentation

Every public function should contain:

- Description
- Mathematical interpretation
- Parameters
- Return value

## Benchmarks

Benchmark results should follow the schema defined in

docs/benchmark-specification.md
